# Państwa Miasta

## Wymagania
- Node.js (wersja 18 lub nowsza)
- npm

## Uruchamianie lokalne

Aby uruchomić projekt lokalnie, musisz uruchomić osobno serwer (backend) oraz aplikację kliencką (frontend).

### 1. Serwer (Backend)

```bash
cd backendTest
npm install
node index.js
```
Serwer domyślnie nasłuchuje na porcie `3000`.

### 2. Aplikacja (Frontend)

```bash
cd frontend
npm install
npm run dev
```
Aplikacja domyślnie będzie dostępna pod adresem: [http://localhost:5173](http://localhost:5173)
